# Project Name: nft_elec5620
## Working functionalities of the project
- Registration
- Login/Login out
- Mint NFT
- Update NFT
  * modify information
  * set sell
- View market/detail
- ShoppingCart
  * add NFT
  * delete NFT
  * payment
- Search 
  * keywords
  * price
- View purchase record
- View wallet balance
- View my NFTs
